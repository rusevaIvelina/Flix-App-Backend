MyFlix Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Returns a list of all movies to the user /movies READ None A Json array holding data about all movies from the list
Returns data about a single movie by title /movies/:title READ None JSON object holding data about a single movie:
Example:{ title: 'Die Hard', genre: 'action', releaseYear: '1988', academyAwards: '4', director: { name: 'John McTiernan', birthYear: '1951', birthPlace: 'Albany, NY', education: 'American Film Institute', deathYear: 'n/a' }
Retruns genre description by movie title (e.g., "Crime") /movies/genre/:name READ None Returns an array of all JSON movie objects within the database that includes the genre passed in the url parameter
Returns data about the movie director (e.g., "Martin Scorsese") /movies/director/:name READ None Returns a JSON object containing data of the director based on the movie name that was passed into the url parameter
Allows new users to register /users CREATE None A JSON object holding data about a new user
Example:{ Username: "alexp", Password: "1234", Email: "alexp@gmail.com", Birthday: "10/01/1966" }
Returns data of all users /users READ None A JSON object hodling data of each user
Allows users to update their info by username /users/:username UPDATE A Json object holding data about the updated user info Returns a message confirming that the user info was successfully updated
Example: "Your username and password were successfully updated."
Allows users to add a movie to their profile /users/:username/movies/:MovieID CREATE A JSON object hodling data about a single movie Returns a message confirming the user successfully added a new movie to their profile.
Example: "Scarface was successfully added to your profile."
Allow users to remove a movie from their profile /users/:username/movies/:MovieID DELETE None Returns a message confirming the movie was successfully deleted from the user's profile.
Example: "Scarface was successfully removed from your profile."
Allow existing users to disregiter /users/:username DELETE None Returns a message confirming the user successfully deleted their account
Example: "Your account was successfully deleted."
